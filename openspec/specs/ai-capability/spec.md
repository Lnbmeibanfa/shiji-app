# AI能力规范（ai-capability）

本规范定义食迹项目中 AI 能力的基本原则。

---

## 需求：AI能力由后端统一管理

AI调用必须由后端统一管理。

### 场景：用户请求分析

- 客户端提交文本或图片
- 后端负责拼接 prompt
- 后端调用 AI API
- 后端返回分析结果

---

## 需求：AI提示词必须版本化管理

AI提示词必须存储在代码仓库中。

### 场景：新增 AI 功能

- prompt 必须写入 prompts 文件
- 不允许直接写死在代码中

---

## 需求：AI必须具备安全兜底

AI输出必须经过安全处理。

### 场景：AI返回异常内容

- 后端必须进行内容过滤
- 必须提供兜底提示
